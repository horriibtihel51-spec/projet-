# Medical SOA Frontend

Application React pour la gestion médicale SOA.

## Installation

1. Cloner le repository
2. `npm install`
3. `npm start`

## Structure

- `/src/components` - Composants React
- `/src/services` - Services API
- `/src/hooks` - Hooks personnalisés
- `/src/styles` - Feuilles de style
- `/src/utils` - Utilitaires

## API Backend

L'application nécessite le backend sur http://localhost:3001
